# 《高中生下课后的迷乱生活》小说大纲

## 1. 核心信息
- **题材**：青春欲孽 / 肉体摩擦 / 欲望交织
- **风格**：底层审美、直接粗犷、快节奏爽文
- **核心冲突**：原始兽欲与校园文明的剧烈碰撞，清纯女神与高冷教师的彻底堕落。
- **章节数**：15章
- **字数要求**：每章3000字以上

## 2. 角色列表
- **阿强**：高二体育生，全书欲望引擎，野蛮征服者。
- **林晓晓**：英语课代表，全校女神，堕落核心。
- **王伟**：富二代，阴险推手，利用金钱与视频玩弄人心。
- **苏曼**：实习英语老师，高冷外表下的放荡灵魂。
- **陈教官**：器材室管理者，视频要挟者，幕后猎食者。
- **苏晴**：副校长之女，学生会主席，背德快感的追求者。
- **校长**：满口道德的伪君子，最终沦为肉奴。

- [completed] **第1章：放学后的隐秘角落** - 燥热校园，阿强诱骗林晓晓进入器材室，初步展现暴力美学与肉欲萌芽。
- [completed] **第2章：体育器材室的汗水** - 狂暴的第一次。阿强在发霉垫子上强占林晓晓，女神尊严碎裂，原始欲望觉醒。
- [completed] **第3章：图书馆的深夜燥热** - 王伟登场，利用发卡要挟林晓晓在旧书库苟合。阿强闯入，形成初步的肉体竞争。
- [completed] **第4章：所谓“补课小组”的真相** - 教室内的淫靡。阿强、王伟、林晓晓、刘芳在讲台与课桌间的混战，规则彻底崩塌。
- [completed] **第5章：办公室里的禁果** - 教师沦陷。阿强在教研室强干实习老师苏曼，权力关系的倒置。
- [completed] **第6章：野外郊游的狂欢** - 青云山野营。篝火旁的四人乱交，陈教官在暗处偷拍。

- [completed] **第8章：空屋派对的癫狂** - 王伟豪宅。苏曼老师被诱骗至此加入混战，苏晴身份伏笔。
- [completed] **第9章：暴风雨后的余波** - 陈教官利用全套视频控制阿强与王伟。体育器材室旧仓库内，陈教官在发霉垫子上强占林晓晓，阿强与王伟沦为帮凶。
