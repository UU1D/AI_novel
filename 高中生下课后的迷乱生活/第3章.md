# 第3章：图书馆的深夜燥热

临江一中的图书馆，是一栋建了三四十年的老红砖破楼。入夜后，那几盏昏黄、忽明忽暗的路灯照在爬满了枯死藤蔓的外墙上，那些黑黢黢、张牙舞爪的影子在那儿随着夜风乱晃，活像一群索命的鬼影。比起操场那边没日没夜的吵闹，这儿本来该是这烂学校里最安静的地儿，可在这闷热得让人浑身冒油、嗓子眼发干的深夜，死静里却透着股子让人脊背发凉、心里直打鼓的邪气。

二楼阅览室里，那台老掉牙的中央空调在那儿“嗡嗡”乱响，震得窗户玻璃都在跟着打哆嗦，听起来跟快断气的老头子在那儿倒气儿没两样。虽然冷风口在那儿断断续续喷着一点凉气，但屋子里还是那股子怎么也散不掉的烂纸味，还有那种长年不见阳光、让人鼻尖发酸、脑袋发晕的霉味。

这股子味道钻进鼻孔，憋得人胸口发闷，脑袋里跟塞了一团乱麻似的。

林晓晓一个人孤零零地坐在靠窗的那个角落里，面前摊着本厚得跟砖头没两样的英语真题。台灯那惨白、刺眼的光照在她脸上，白得跟纸没两样，没一点血色。一个小时过去了，她那双布满了血丝的眼死死盯着那一页，一个单词儿也没看进去，脑子里全是昨天在器材室发生的那些肮脏画面。

她的下面，现在正难受得要命，像是有一团火在烧。

昨天在那个闷热、腥臊得让人反胃的器材室里，被阿强那个蛮横得像头野猪、浑身汗臭味的蛮牛狠狠折腾了整整两个多小时，她的大腿根现在还火辣辣地疼，每走一步都觉得胯下像是在被针扎。尤其是那个最嫩的地方，被阿强那根铁棍硬生生捅开后的那种胀痛和火烧火燎的劲儿，无时无刻不在提醒她：她这个全校公认的班花，已经在那堆发霉、满是尘土的垫子上，彻底烂透了，成了一个被人玩过的破鞋。

更让他脊背发凉的是，从进图书馆开始，她就总觉得有一双发红、贪婪、像是要把她衣服扒光的眼珠子，正躲在那些黑漆漆、透着死气的书架后面，死死盯着她的一举一动。那种感觉，就像是有条黏糊糊、带着冷气的毒蛇顺着脚踝往上爬，惊得她浑身直起鸡皮疙瘩，连大气儿都不敢喘。

“林晓晓，这么晚了还在这儿装勤快呢？这想定力，我是真佩服得五体投地啊。”

一个阴阳怪气、带着股子让人不舒服的邪劲儿的声音，突然在林晓晓后脑勺那儿响了起来。

林晓晓吓得浑身猛地一抖，手里攥着的钢笔“啪”地一声掉在了桌上，在那儿滚了好几圈。她猛地回过回，看见王伟正歪着身子靠在旁边的书架上，似笑非笑、眼神里全是下流淫光地盯着她看。

王伟是学校里有名的富二代，整天穿着身名牌，金丝眼镜后面那双眼，总透着股子让人脊背发凉的贪婪劲儿。他家在这临江市有权有势，平日里在这学校里横着走，连老师都得让他三分。

“王……王伟？你还没走？”林晓晓强压着心里的慌乱，声音在那儿打着颤，两只手死死抓着校服裙摆。

“走？这么带劲的晚上，我哪舍得走啊？”王伟一边说着，一边大摇大摆地坐到了林晓晓对面。

他根本就没带书，双手抱在胸前，那双眼像钩子一样，在林晓晓那张红通通、由于惊吓而冒汗的小脸上乱扫。最后，他的目光死死钉在了林晓晓校服领口那儿。

虽然领口扣得死紧，但林晓晓一紧张，呼吸就急促得厉害，胸口那一对肉球由于剧烈起伏而显得格外勾人。领口松动间，那一抹还没散掉、青紫交加的吻痕，在惨白的灯光下晃了一下，刺眼得很。

林晓晓吓得脸色发白，赶紧伸手捂住领口，眼神里全是惊恐。

“林晓晓，你昨天放学干啥去了？我路过体育器材室的时候，听见里边动静大得很啊，那叫声……啧啧，叫得那叫一个浪，跟窑子里的婊子没两样。”王伟压低嗓门，语气里全是下流的调戏，在那儿嘿嘿坏笑着。

“我……我不知道你说啥。我昨天放学直接回家了，没去过啥器材室。”林晓晓的心跳快得要撞破胸膛，低着头，手指由于用力都变得惨白。

“是吗？那我在那儿捡到这玩意儿，你怎么说？这全校好像就你有吧？”王伟冷笑一声，从裤兜里掏出一个东西，“啪”地一声拍在了桌子上。

那是一个带着淡淡香味、沾了一点点灰尘的小草莓发卡。

林晓晓脑子“嗡”的一声，整个人都傻了，像是掉进了冰窟窿里。那是她最喜欢的发卡，昨天被阿强那个蛮牛在那儿折腾得太疯，她竟然完全没发现发卡弄丢了。

“王伟，你想咋样？你要钱我给你，只要你别说出去！”林晓晓彻底崩溃了，眼泪在眼眶里直打转，眼珠子里全是哀求。

“钱？你觉得老子缺那点臭钱吗？”王伟狞笑一声，站起身，慢吞吞地走到林晓晓身后。

他那双带着刺鼻、昂贵香水味的大手，猛地按在林晓晓白嫩、滑溜的肩膀上，使捏了捏，感受着那皮肉由于恐惧而产生的剧烈颤抖。

“林晓晓，要是全校那些崇拜你的男生知道，你被阿强那个浑身汗臭、家里穷得揭不开锅的穷鬼按在发霉的垫子上操，那场面，啧啧，想想就让人想射一裤子啊。你说，要是你爸妈知道了，他们还有脸在这临江市待下去吗？”王伟凑在林晓晓耳根子那儿，在那儿喷着一股子带着酒气的腥热气儿。

“求你……求求你别说出去……只要不说，让我干啥都行……”林晓晓的眼泪止不住地往下掉，在那本英语真题集上砸出一个个圆乎乎的湿印子。

“干啥都行？这可是你说的，老子可没逼你。”王伟冷笑一声，语气里全是得逞后的狂妄，“跟老子来，去后边的旧书库。只要今晚把老子伺候爽了，发卡还你，秘密也烂在老子肚子里。”

王伟粗暴地一把拽住林晓晓那纤细、白嫩的胳膊，像拖一条快要断气的死狗一样，蛮横地把她拉进了图书馆最深处那个阴暗、潮湿、到处是烂纸味和灰尘的旧书库。

旧书库里更憋闷，那一排排高大的书架在那儿立着，阴森得很，连个透气的小窗户都没有。王伟猛地一把将林晓晓推倒在一排书架上。

“砰！”

林晓晓的后背重重地撞在坚硬的红木架子上，震得上面掉了一层厚厚的黑土，弄得她那一头秀发全乱了。

“王伟……别在这儿……咱们去外面开房行不……这儿太脏了……”

“操！跟阿强那个穷鬼在器材室那种脏地儿就行，跟老子就不行？你这骚货是不是欠操？”王伟被激怒了，抬手在那张红通通的俏脸上狠狠扇了一个耳光。

“啪！”

那响声在死静、压抑的书库里回荡个没完。林晓晓的半边脸瞬间就红肿了起来，在那儿火辣辣地疼。她呆呆地看着王伟，眼神空洞得像个没了魂的木头娃娃。

王伟急不可耐地扯开了林晓晓的校服领口。

“撕拉——！”

这一次，林晓晓连求饶的劲儿都没了。她任由王伟那双由于兴奋而乱抖的手，在她那还没消掉青紫痕迹、甚至还有几个齿痕的身体上乱摸乱掐，像是在揉捏一团烂泥。

“操，果然被那蛮子干得不轻，胸口全是牙印，真他妈骚。”王伟盯着那些吻痕，眼神里全是病态的嫉妒和快感，“老子今天要让你知道，谁才是这学校真正的主人！阿强算个屁！”

他猛地解开皮带，金属扣在那儿发出一声脆响。裤子堆在脚脖子那儿，露出他那根紫胀、跳动着青筋的东西，带着急不可耐的贪婪欲望，猛地挺了出来。

“林晓晓，把腿张开！张大点！给老子好好看着！”王伟吼道，那声音在那狭小的书库里震得人耳朵发麻。

林晓晓哆嗦着，慢慢分开了那双由于昨天过度劳累还在发颤的修长白腿，露出了下面那片早已由于生理刺激而变得泥泞不堪、正不断往外流水儿的黑草丛。

王伟迫不及待地挤了进去，把林晓晓死死抵在冰冷、坚硬的书架上。那木板咯着林晓晓的脊梁骨，跟王伟那满是冷汗、滚烫的身体撞在一起，冷热交替间，带起一阵阵让她发疯的异样感。

“啊——！”

当王伟那根紫胀的东西猛地、没一点怜悯地捅进那还红肿着的深处时，林晓晓发出一声凄惨的叫声。那种撕裂般的剧疼让她想晕过去，可王伟却疯了似地加快了动作，在那儿横冲直撞。

“啪！啪！啪！啪！啪！”

在那全是霉味、死气沉沉的书库里，瞬间响起了一连串沉重、响亮且极其下流的肉体碰撞声。王伟每一下重击都撞得书架“吱呀吱呀”乱响，上面的书也跟着在那儿乱颤，仿佛随时都会掉下来把两人给埋了。

“林晓晓，爽不爽？阿强那个穷鬼能给你啥？他只会像个畜生一样使蛮力，老子能让你上天！”王伟一边疯了似地抽送，在那泥泞的窄道里乱钻，一边在林晓晓白嫩的脖子上狠狠咬了一口，带出了一圈殷红的血印子。

“呜呜……阿强……阿强……”林晓晓在迷乱和极度惊恐中，由于害怕，竟然下意识地喊出了阿强的名字。

“操！跟老子干，还敢喊那个穷鬼的名字！你这贱货！”王伟彻底炸了，猛地揪住林晓晓的头发，强迫她仰起头，然后更加残暴地在那窄小、湿润的甬道里横冲直撞，带起一阵阵“咕唧咕唧”的粘稠声响，在那寂静的空间里显得格外放肆。

“老子干死你！干死你这不知廉耻、只知道要肉棒的小浪货！”

王伟像头疯了的野狗，在那儿疯狂宣泄。林晓晓的身体在书架上无力地上下颠着，那种疼慢慢变成了一种让她感到绝望的空虚，在那儿不停地流着泪。

就在这时候，书库外的走廊里，突然传来了一串极其细微、却极有节奏的脚步声。

“咔哒。”

那是硬底鞋踩在旧木地板上发出的声音。

王伟猛地停住了动作，僵在那儿，满头大汗地盯着门口。林晓晓也惊恐地瞪大了眼，死死捂住自己的嘴，在那儿屏住呼吸，一个屁都不敢放。

那脚步声越来越近，每一下都像是直接砸在两人的心尖子上，震得人心发慌。

“谁？给老子滚出来！别在那儿装神弄鬼！”王伟压低嗓门，在那儿虚张声势地吼着，手却不由自主地在抖。

没人吭声，只有那脚步声，最后稳稳地停在了旧书库那扇半掩着的木门口。

阅览室那惨白的日光灯闪了一下，一个高大、黝黑、带着一股子压迫感的黑影子，缓缓投在了地板上，拉得老长。

是阿强。

他手里竟然还拎着个满是泥印子的破篮球，身上还是那股子浓烈、让人头晕的汗臭味。他那双泛着红丝的眼里全是冰冷和一种让人胆战心惊的兴奋，死死盯着衣衫不整、满身是汗的王伟和林晓晓。

“伟哥，玩得挺花啊，连这儿都能翻出来。怎么，在这儿吃独食呢？也不叫上兄弟我一起过来爽爽？”

阿强那粗鲁、浑浊、带着股子土腥味的声音在书库里响起，瞬间就震碎了这深夜图书馆最后的一点死静。

林晓晓看着这两个此时都满脸欲望、像盯着一块肥肉一样盯着自己的男人，心里的最后一块阵地彻底没了。她在那儿无力地瘫软在书架旁，眼泪流干了，心里只剩下一个念头：她已经彻底成了这群校园野兽共同的玩物，再也回不去那个曾经圣洁的世界了。

“阿强……你来得正好，这骚货刚才还喊你名字呢，看来是想让你也加入啊。”王伟狞笑一声，眼里闪过一丝变态的快感。

“是吗？既然班花这么想老子，那老子今天就当个助兴的！”

阿强把篮球往旁边一扔，大步跨了过来。在那昏暗、充满了霉味和肉欲气息的旧书库里，三具年轻、却早已腐烂到了骨子里的肉体，在那儿紧紧纠缠在一起，拉开了这场关于毁灭与欲望的更大序幕。

窗外，月亮被云彩遮住了，整个临江一中都被黑暗死死吞没。只有那破旧图书馆的旧书库里，还不时传来阵阵让人面红耳赤、极其下流的撞击声和呻吟声，在那儿回荡个不停。
